  • First app

This tutorial assumes you are familiar with modern Javascript and its ecosystem (e.g. you know how and when to use tools like npm, Webpack, etc.), know how to write regular Web Apps, and are interested in building one that stores data to a Solid Pod.

It does not assume prior knowledge of Semantic Web technologies, Linked Data, RDF, or knowing what those terms mean in the first place.

We will be building a note-taking app that can add notes to someone’s Pod, and can read existing ones. If you’re so inclined, you can play around with the source code of the finished app already.
